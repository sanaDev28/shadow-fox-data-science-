# Task 2: Sentiment Analysis on X (Twitter) Using VADER

## Overview

This project focuses on analyzing the sentiment of posts from X (formerly Twitter) using the VADER Sentiment Analyzer. The objective is to identify whether posts express positive, negative, or neutral sentiments and to understand the overall mood of conversations within the dataset.

VADER (Valence Aware Dictionary and sEntiment Reasoner) is a lexicon and rule-based sentiment analysis tool specifically designed for social media text. It is effective in handling informal language, emojis, abbreviations, and expressions commonly found on social networking platforms.

## Objectives

* Perform sentiment analysis on X posts using VADER.
* Classify posts into Positive, Negative, and Neutral categories.
* Measure the distribution of sentiments in the dataset.
* Understand the overall sentiment trend of the conversation.

## Approach

* Loaded and explored the dataset obtained from Kaggle.
* Cleaned and prepared the text data for analysis.
* Applied the VADER Sentiment Analyzer to each post.
* Categorized posts based on sentiment scores.
* Analyzed the number of positive, negative, and neutral posts.
* Visualized the sentiment distribution for better understanding.

## Results

The sentiment analysis provided insights into the emotional tone of the discussions. By comparing the counts of positive and negative posts, it became possible to identify whether the conversation was generally supportive, critical, or neutral.

## Conclusion

VADER proved to be an effective tool for sentiment analysis on social media data. The analysis helped in understanding user opinions and overall sentiment trends within the dataset. Such techniques can be useful for monitoring public opinion, detecting negative behavior, and gaining insights from online discussions.


**Tools Used:Python, Pandas, NLTK, VADER Sentiment Analyzer, Matplotlib, Seaborn.
